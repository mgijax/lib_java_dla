OK_TO_PREVENT_ANNOTLOAD_UPDATE

Controls whether or not to bcp the annotation results into the database. The default is false.

 

ANNOTLOAD_PATH

The name of the path conating the annotload python program. The default value is /usr/local/mgi/live/dataload/annotload/annotload.py.

 

ANNOTLOAD_MODE

The mode of the annotload. See annotload documentation for more information. The default value is `NEW'.

 

ANNOTLOAD_TYPE

The name of the annotation as stored in the VOC_Annot table. There is no default value and its value must be set to avoid an exception..

 

ANNOTLOAD_JNUMBER

The jnumber for the annotload. There is no default value and its value must be set to avoid an exception..

 

VOCLOAD_COMMAND_PATH

The pathname of the vocload. The defualt value is /usr/local/mgi/live/dataload/vocload/simpleLoad.py

 

VOCLOAD_RCD_FILENAME

The name of the rcd file to use. The default is /usr/local/mgi/live/dataload/annotload/simple.rcd.

 

VOCLOAD_OK_TO_PREVENT_UPDATE

Controls whether or not to bcp the vovload results into the database. The default is false.

 

VOCLOAD_IS_SIMPLE

Determines whether or not this is a simple vocload. The default is true.

 

VOCLOAD_IS_FULL

Determines whether or not this is a full vocload. The default is true.

 

VOCLOAD_NAME

Determines the name of the vocabulary. This value has no default and must be set by the application.

VOCLOAD_OUTPUTDIR

Determines the name of the output results path. The default is the current runtime directory.
